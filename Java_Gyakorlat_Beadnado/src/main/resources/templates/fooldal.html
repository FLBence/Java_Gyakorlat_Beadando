Üdvözlünk a Forma–1 Adatbázis Alkalmazásban!

Főoldal

Ez a webalkalmazás a Forma–1 világához kapcsolódó adatok kezelését és megtekintését teszi lehetővé. A rendszer különböző funkciókat biztosít a felhasználók és adminisztrátorok számára.

Funkciók

  • Pilóták adatainak megtekintése
  • Felhasználói regisztráció és bejelentkezés
  • Üzenetek küldése regisztrált felhasználóknak
  • Admin funkciók (adatkezelés, törlés, módosítás)
  • Diagram a pilóták adataival kapcsolatban
  • CRUD menü adatok frissen tartása érdekében

Adatbázis

  • Pilóták, GP-k és az eredmények adatai
  • Felhasználók és szerepköreik
  • Üzenetek

Felhasznált technológiák

  • Java 21
  • Spring Boot, Spring MVC, Spring Security
  • Thymeleaf templaterendszer
  • MySQL adatbázis
  • JPA / Hibernate
  • HTML, CSS, JS, Bootstrap
